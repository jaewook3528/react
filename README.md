https://www.mongodb.com/try/download/community 에서 mongo DB 다운로드 후 설치 
git 소소 다운로드

cd blog-backend 백엔드 디렉토리로 이동 후
bash 창에서 yarn start:dev 실행

bash 창에서 yarn start 실행
